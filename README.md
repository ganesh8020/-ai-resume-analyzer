# AI Resume Analyzer

This is a simple AI-based resume analyzer that extracts key skills and generates a summary from resume text.

## Features
- Extracts important skills from resume
- Generates a short summary
- Simple and interactive UI using Streamlit

## Tech Stack
- Python
- Streamlit

## How to Run

1. Clone the repository
2. Install dependencies:
   pip3 install streamlit

3. Run the app:
   streamlit run app.py

## Notes
This is a lightweight version built using rule-based logic to simulate AI behavior for resume analysis.

## API Setup

If using OpenAI API, set your API key as an environment variable:

export OPENAI_API_KEY="your_api_key"
